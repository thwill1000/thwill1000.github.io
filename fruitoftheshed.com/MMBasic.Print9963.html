The original program was written for the Colour Maximite running MMBasic 4.5. It utilised current stunts in the earth leg for input (charging) current, load current and calculated battery current (charging or discharging).
A new version running on Micromites under mmbasic 5.04 and utilising ACS758 high side current sense modules is in final beta and should be published here in a few weeks.
