Welcome to the MicroMite ArmMite and MMX Hardware namespace of Fruit Of The Shed!

This is the main page of the namespace.

Use this area for hardware info relating specifically to the MicroMite Mk2, MicroMite eXtreme using MicroChips's PIC32MX & PIC32MZ controllers, and the ArmMite F4 & H7 using ST ARM Cortex controllers.

For code to run on these devices, please see the MMBasic zone.
