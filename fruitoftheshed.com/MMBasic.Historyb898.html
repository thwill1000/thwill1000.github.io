A small wrapper function for Quazee137's tiny Day Of The week function. Takes a date string as dd-mm-yyyy e.g. that returned by DATE$ or LEFT$(Humantime$,10) and returns the day name.

Beware the date format must be dd-mm-yyyy, no error checking is performed.


Syntax:
DoWDay$(date_str$)

Example Usage:
Day$=DoWDay$("20-08-1995")

Dependencies:
  • Quick Day Of Week (and tiny too)

	Function DoWDay$(dd$)
		Select Case DoW(val(mid$(dd$,4,2)),val(left$(dd$,2)),val(right$(dd$,4)))
			Case 1:DoWDay$="Sun"
			Case 2:DoWDay$="Mon"
			Case 3:DoWDay$="Tue"
			Case 4:DoWDay$="Wed"
			Case 5:DoWDay$="Thu"
			Case 6:DoWDay$="Fri"
			Case 7:DoWDay$="Sat"
		End Select
	End Function
