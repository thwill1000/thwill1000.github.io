Colour Maximite 2

Modified on 2020/07/24 21:22 by CaptainBoing — Categorized as: Uncategorized

The Colour Maximite 2 uses a newer version of MMBasic with some new and different commands, sufficiently different from other *mites that it warrants its own zone which can be found here.

Please feel free to add new pages and contribute

Many of the commands are documented and the standard MMBasic language references should be used along with the additional documents for the CMM2 specific functionality.

There are some examples in the forum thread at https://www.thebackshed.com/forum/ViewTopic.php?FID=16&TID=12378&LastEntry=Y#150791
